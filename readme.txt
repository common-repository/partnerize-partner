== Description ==
This will add a settings page to the Admin dashboard so you can provide your Partnerize API credentials. The plugin will then query the Partnerize API to obtain all of your available, and approved, partner marketing campaigns. When you add a new post to the site there will be an option in Editor to select your tracking link and add any of the available parameters to the URL.

== Installation ==
Installation of this plugin is easy. Just activate the plugin and visit the Settings page to add your API credentials.
